DEPLOYMENT SEQUENCE
===================
1. Create HuggingFace Space
2. Deploy backend (push to HuggingFace)
3. Configure secrets in HuggingFace Spaces
4. Test backend endpoints
5. Update frontend API configuration
6. Replace ChatWidget.jsx
7. Deploy frontend (npm run deploy)
8. Test integration end-to-end
